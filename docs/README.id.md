## Daftar Fitur

### Latihan Kata

- Mode latihan: Mengikuti / Dikte / Tes mandiri / Menulis dari ingatan
- Mode cerdas: Menghitung kata-kata pembelajaran secara otomatis berdasarkan kurva memori, memperdalam ingatan melalui dikte
- Mode bebas: Tanpa batasan, rencanakan pembelajaran Anda sendiri
- Menyediakan fonetik, pengucapan (Amerika/Inggris), contoh kalimat, frasa, sinonim, kata berakar sama, etimologi, statistik kesalahan, dan lainnya

### Menghafal Artikel

- Buku teks klasik bawaan; Anda juga dapat menambah atau mengimpor artikel dengan terjemahan satu klik dan perbandingan bilingual
- Mode ganda mengikuti + dikte, input kalimat per kalimat dengan pengucapan otomatis untuk menghafal lebih efisien
- Mendukung mendengarkan sambil menulis dari ingatan untuk memperkuat pembelajaran

### Favorit, Kata Salah, Dikuasai

- Kata-kata yang salah diketik selama belajar secara otomatis ditambahkan ke buku kata salah Anda untuk ditinjau nanti
- Aktif menambahkan kata ke dikuasai untuk secara otomatis melewatinya di sesi mendatang
- Tambahkan kata ke favorit untuk konsolidasi dan tinjauan

### Sangat Dapat Disesuaikan

- Efek suara keyboard yang kaya
- Pintasan yang dapat disesuaikan
- Opsi pengaturan yang sangat dapat dikonfigurasi

### Bersih dan Efisien

- Desain bersih, UI modern, tanpa iklan
- Antarmuka yang menyegarkan, operasi sederhana
- Tidak ada langganan paksa ke platform manapun

### Pustaka Kosakata

Kosakata yang umum digunakan bawaan termasuk CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, bahasa Inggris pascasarjana, TEM-4, TEM-8, dan lainnya.
Dirancang untuk memenuhi kebutuhan pembelajaran kosakata sebagian besar pengguna. Kontribusi kosakata tambahan dari komunitas disambut.

## Menjalankan Proyek

#### Catatan: Proyek ini dapat berjalan mandiri dengan data disimpan secara lokal. Pencadangan manual diperlukan saat mengganti perangkat; ini tidak memengaruhi penggunaan normal.
Proyek ini dibangun dengan `Nuxt` dan membutuhkan lingkungan Node.js.

1. Instal NodeJS, lihat [dokumentasi resmi](https://nodejs.org/en/download)
2. Proyek ini besar. Disarankan menggunakan `git clone --depth 1 https://github.com/zyronon/TypeWords.git` untuk hanya mengkloning commit terbaru. Fungsi Download ZIP GitHub tidak akan berfungsi dengan benar.
3. Di direktori root proyek, buka terminal dan jalankan `pnpm install` untuk mengunduh dependensi.
4. Jalankan `pnpm run dev` untuk memulai proyek. Alamat default adalah [`http://localhost:5567`](http://localhost:5567)
5. Buka [`http://localhost:5567`](http://localhost:5567) di browser Anda untuk mengakses proyek.
6. Jalankan `pnpm run generate` untuk membangun file proyek.
