## Lista de funciones

### Práctica de palabras

- Modos de práctica: Seguimiento / Dictado / Autoevaluación / Escritura de memoria
- Modo inteligente: Calcula automáticamente las palabras de aprendizaje basándose en curvas de memoria, profundizando la memoria a través del dictado
- Modo libre: Sin restricciones, planifica tu propio aprendizaje
- Proporciona fonética, pronunciación (americana/británica), oraciones de ejemplo, frases, sinónimos, palabras con la misma raíz, etimología, estadísticas de errores y más

### Memorización de artículos

- Libros de texto clásicos incorporados; también puedes agregar o importar artículos con traducción con un clic y comparación bilingüe
- Modos duales de seguimiento + dictado, entrada oración por oración con pronunciación automática para una memorización más eficiente
- Soporta escuchar mientras escribes de memoria para reforzar el aprendizaje

### Favoritos, Palabras incorrectas, Dominadas

- Las palabras escritas incorrectamente durante el aprendizaje se agregan automáticamente a tu libro de palabras incorrectas para revisión posterior
- Agrega activamente palabras a dominadas para omitirlas automáticamente en sesiones futuras
- Agrega palabras a favoritos para consolidar y revisar

### Altamente personalizable

- Ricos efectos de sonido de teclado
- Atajos personalizables
- Opciones de configuración altamente configurables

### Limpio y eficiente

- Diseño limpio, interfaz moderna, sin anuncios
- Interfaz refrescante, operación simple
- Sin suscripción forzada a ninguna plataforma

### Biblioteca de vocabulario

Vocabulario de uso común incorporado incluyendo CET-4, CET-6, GMAT, GRE, IELTS, SAT, TOEFL, inglés para posgrado, TEM-4, TEM-8 y más.
Diseñado para satisfacer las necesidades de aprendizaje de vocabulario de la mayoría de los usuarios. Se aceptan contribuciones de vocabulario adicional de la comunidad.

## Ejecutar el proyecto

#### Nota: Este proyecto puede ejecutarse de forma independiente con datos guardados localmente. Se requiere respaldo manual al cambiar de dispositivos; esto no afecta el uso normal.
Este proyecto está construido con `Nuxt` y requiere un entorno Node.js.

1. Instala NodeJS, consulta la [documentación oficial](https://nodejs.org/en/download)
2. El proyecto es grande. Se recomienda usar `git clone --depth 1 https://github.com/zyronon/TypeWords.git` para clonar solo el último commit. La función Download ZIP de GitHub no funcionará correctamente.
3. En el directorio raíz del proyecto, abre una terminal y ejecuta `pnpm install` para descargar las dependencias.
4. Ejecuta `pnpm run dev` para iniciar el proyecto. La dirección predeterminada es [`http://localhost:5567`](http://localhost:5567)
5. Abre [`http://localhost:5567`](http://localhost:5567) en tu navegador para acceder al proyecto.
6. Ejecuta `pnpm run generate` para compilar los archivos del proyecto.
