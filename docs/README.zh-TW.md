## 功能列表

### 單詞練習

- 練習模式：跟寫 / 聽寫 / 自測 / 默寫
- 智慧模式：記憶曲線自動計算學習單詞，並通過默寫加深記憶
- 自由模式：不受限制，自行規劃
- 提供音標、發音（美音、英音）、例句、短語、近義詞、同根詞、詞源、錯誤統計等功能

### 背文章

- 內建經典教材書籍，也可自行添加、匯入文章，提供一鍵翻譯、譯文對照功能
- 跟打 + 默寫雙模式，逐句輸入，自動發音，讓背誦更高效
- 支援邊聽邊默寫，強化記憶

### 收藏、錯詞本、已掌握

- 學習單詞時輸入錯誤自動添加到錯詞本，方便後續複習
- 可主動添加到已掌握，後續學習時自動跳過
- 可主動添加到收藏中，以便鞏固複習

### 高度自由

- 豐富的鍵盤音效
- 可自訂快捷鍵
- 高度客製化的設定選項

### 簡潔高效

- 簡潔設計，現代化UI，無廣告
- 介面清爽，操作簡單
- 不強制關注任何平台

### 詞庫

內建了常用的 CET-4、CET-6、GMAT、GRE、IELTS、SAT、TOEFL、考研英語、專業四級英語、專業八級英語等詞庫。
盡可能滿足大部分使用者對背單詞的需求，也非常歡迎社群貢獻更多的詞庫。

## 運行

#### 注：本專案可單獨運行，資料保存在本地，換設備需手動備份資料，不影響正常使用；
本專案是基於 `Nuxt` 開發的，需要 Node 環境來運行。

1. 安裝 NodeJS，參考[官方文件](https://nodejs.org/en/download)
2. 專案檔案很大，推薦使用 `git clone --depth 1 https://github.com/zyronon/TypeWords.git` 命令只克隆最近一次提交。直接下載 Github 提供的 Download ZIP 功能是無法運行的
3. 在專案根目錄下，打開命令列，運行 `pnpm install` 來下載依賴。
4. 執行 `pnpm run dev` 來啟動專案，專案預設地址為 [`http://localhost:5567`](http://localhost:5567)
5. 在瀏覽器中打開 [`http://localhost:5567`](http://localhost:5567) 來訪問專案。
6. 執行 `pnpm run generate` 打包專案檔案
